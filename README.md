# Parking Lot Management System

##  Overview
A Spring Boot-based backend system designed to automate parking operations. This project demonstrates the application of Java backend principles learned during my **QSpiders** and **KodNest** training.

##  Tech Stack
- **Language:** Java 17
- **Framework:** Spring Boot 3.x
- **ORM:** Hibernate / Spring Data JPA
- **Database:** MySQL
- **Architecture:** REST API

## Key Features
- **Automated Slot Allocation:** Efficiently assigns parking spots based on availability.
- **Dynamic Fee Calculation:** Logic implemented to charge users based on time-duration stay.
- **REST Endpoints:** Clean APIs for vehicle entry and exit processing.

While this project focuses on system design, my algorithmic foundation is built on solving **500+ problems** across LeetCode and CodeChef, allowing for optimized logic in the backend services.
